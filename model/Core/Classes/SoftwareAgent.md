# SoftwareAgent

## Summary

A software agent.

## Description

A SoftwareAgent is a software program that is given the authority (similar to a
user's authority) to act on a system.

## Metadata

- name: SoftwareAgent
- SubclassOf: Agent
- Instantiability: Concrete
