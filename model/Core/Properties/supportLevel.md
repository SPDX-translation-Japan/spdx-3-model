# supportLevel

## Summary

Specifies the level of support associated with an artifact.

## Description

supportLevel provides an indication of what support expectations that the
supplier of an artifact is providing to the user.

## Metadata

- name: supportLevel
- Nature: ObjectProperty
- Range: SupportType

## Summary @ja

アーティファクトに関連付けられたサポートの水準を指定する。

## Description @ja

supportLevel は、アーティファクトの供給者が利用者に対して提供するサポートに関する期待度合いを示す。この情報は、例えば保守、更新対応、品質保証、問い合わせ対応など、アーティファクトに付随するサポートの範囲や程度を明確にするために使用される。
