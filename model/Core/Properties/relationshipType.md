# relationshipType

## Summary

Information about the relationship between two Elements.

## Description

This field provides information about the relationship between two Elements.

For example, you can represent a relationship between two different Files,
between a Package and a File, between two Packages, or between one SpdxDocument
and another SpdxDocument.

## Metadata

- name: relationshipType
- Nature: ObjectProperty
- Range: RelationshipType

## Summary @ja

二つの Element 間の関係に関する情報。

## Description @ja

このフィールドは二つの Element 間の関係に関する情報を提供する。  
例えば、二つの異なる File 間、Package と File の間、二つの Package 間、または一つの SpdxDocument と別の SpdxDocument の間の関係を表すことができる。
