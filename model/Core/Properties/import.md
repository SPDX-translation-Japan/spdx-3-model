# import

## Summary

Provides an ExternalMap of Element identifiers.

## Description

An import provides an ExternalMap of an Element identifier that is used within a
document but defined external to that document.

## Metadata

- name: import
- Nature: ObjectProperty
- Range: ExternalMap
