# key

## Summary

A key used in a generic key-value pair.

## Description

A key used in generic a key-value pair.

A key-value pair can be used to implement a dictionary which associates a key
with a value.

## Metadata

- name: key
- Nature: DataProperty
- Range: xsd:string
