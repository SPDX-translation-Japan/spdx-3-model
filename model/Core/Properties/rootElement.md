# rootElement

## Summary

This property is used to denote the root Element(s) of a tree of elements contained in a BOM.

## Description

This property is used to denote the root Element(s) of a tree of elements contained in a BOM.
The tree consists of other elements directly and indirectly related through properties or Relationships from the root.

## Metadata

- name: rootElement
- Nature: ObjectProperty
- Range: Element

## Summary @ja

BOM に含まれる要素ツリーのルート要素（群）を示すプロパティ。

## Description @ja

このプロパティは、BOM に含まれる要素ツリーのルート要素（群）を示す。このツリーは、ルートからプロパティや Relationship を介して直接または間接に関連付けられた他の要素によって構成される。
