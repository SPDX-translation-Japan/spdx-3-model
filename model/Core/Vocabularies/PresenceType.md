# PresenceType

## Summary

Categories of presence or absence.

## Description

This type is used to indicate if a given field is present or absent or unknown.

## Metadata

- name: PresenceType

## Entries

- yes: Indicates presence of the field.
- no: Indicates absence of the field.
- noAssertion: Makes no assertion about the field.

## Summary @ja

存在または非存在の区分。

## Description @ja

この型は、与えられたフィールドが存在するのか、存在しないのか、あるいは不明なのかを示すために使用される。

## Entries @ja

- yes: フィールドが存在することを示す。  
- no: フィールドが存在しないことを示す。  
- noAssertion: フィールドについて何も主張しないことを示す。
